### Sing View & UnSing View - _difference_
<br>

<strong>_Sing View_</strong>
* 특정 지역을 검색하여 마커 표출
* 옵션에 따른 마커의 색깔 변화
* 다이얼로그 이용하여 도움말 표시
<br>

<strong>_UnSing View_</strong>
* 키워드로 검색을 하여 주위의 마커 표출
* 옵션에 따른 마커 삭제
* 마커의 제한을 10개로 주고 회원가입을 유도
